# 설치테스트

* 프로젝트 폴더 생성

```
mkdir -p ~/go/src/hello
```

* 소스작성

```go
package main

import "fmt"

func main() {
    fmt.Printf("Hello, World!\n")
}
```

* 빌드

```
go build
```

* 실행

```
./hello
```
